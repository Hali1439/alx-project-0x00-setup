# ALX Project 0x00

This is a starter project scaffolded using `npx create-next-app@latest` with TypeScript support.

## Project Setup

- Framework: [Next.js](https://nextjs.org/)
- Language: TypeScript
- Style: Tailwind CSS
- Linting: ESLint
- Aliases: Configured
- No `/src` directory
- No App Router

## Development Server

To start the server on port 3000:

```bash
npm run dev -- -p 3000
